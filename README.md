## Description

This project is a web-based application built with React and TypeScript, leveraging the Google Gemini AI API to provide intelligent assistance for career pivoting and professional development. It serves as an example of integrating powerful AI capabilities into a modern front-end environment, offering users a smart companion for navigating their professional journey.

## Features

*   AI-powered career guidance and insights
*   Interactive and responsive user interface built with React
*   Seamless integration with Google Gemini API for advanced AI functionalities
*   Modern development setup with Vite for fast build times and development experience
*   TypeScript for robust and maintainable code

## Tech Stack

*   **Frontend:** React, TypeScript, Vite
*   **Backend (API):** Google Gemini API (accessed client-side or via a proxy)
*   **Runtime:** Node.js (for development and package management)

## Installation

To get this project up and running on your local machine, follow these steps:

### Prerequisites

Ensure you have Node.js installed on your system. We recommend using the latest LTS version.

### Steps

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/gemini-ai-career-assistant.git
    ```

    Replace `your-username` with your GitHub username or the organization's name.

2.  **Navigate to the project directory:**

    ```bash
    cd gemini-ai-career-assistant
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Configure your Gemini API Key:**

    Create a `.env.local` file in the root of the project and add your Google Gemini API key:

    ```
    VITE_GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
    ```

    Replace `YOUR_GEMINI_API_KEY` with your actual API key obtained from Google AI Studio.

## Usage

To run the application in development mode:

1.  **Start the development server:**

    ```bash
    npm run dev
    ```

2.  **Access the application:**

    Open your web browser and navigate to `http://localhost:5173` (or the port indicated in your terminal).
